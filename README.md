# Portfolio Angular

## Structure du site

Mon site est composé de deux parties :

- __Public__ :
  - Accueil,
  - A Propos,
  - CV :
    - Compétences,
    - Certifications,
    - Expériences,
    - Parcours scolaire,
  - Projets,
  - Contact,
- __Blog__ :
  - Entreprises,
  - Articles.
